# general-purpose normal-mixture deviate generator: takes *matrices* of probabilities, means, sds

general-purpose normal-mixture deviate generator: takes *matrices* of
probabilities, means, sds

## Usage

``` r
rnormmix0(n, probs, muvals, sdvals)
```

## Arguments

- n:

  number of observations

- probs:

  mixture proportions

- muvals:

  values for the mean

- sdvals:

  values for the standard deviation

## Value

simulations
