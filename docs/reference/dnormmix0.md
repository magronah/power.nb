# Density function for the mixture of Gaussian distributions

takes pars as three vectors (prob, mean, sd), on constrained scale

## Usage

``` r
dnormmix0(x, probs, muvals, sdvals, log = FALSE)
```

## Arguments

- x:

  vector

- probs:

  mixture proportions

- muvals:

  values of the mean

- sdvals:

  values for standard deviataions

- log:

  log scale

## Value

likelihood
