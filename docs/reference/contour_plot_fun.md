# Contour plot for showing predicted power

Contour plot for showing predicted power

## Usage

``` r
contour_plot_fun(
  combined_data,
  power_estimate,
  cont_breaks,
  multiple_samples = FALSE
)
```

## Arguments

- combined_data:

  data used for fitting gam

- power_estimate:

  predicted power

- cont_breaks:

  breaks for contour plot

- multiple_samples:

  not currently used

## Value

ggplot2 object
